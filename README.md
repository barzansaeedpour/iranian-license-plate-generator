# Iranian-license-plate-generator

You can use this project to produce quality Iranian license plates.

از این پروژه برای تولید پلاک های ایرانی با کیفیت می توانید استفاده کنید.

## How to use

- ```
    pip install -r requirements.txt
    ```
- ```
    python main.py 
    ```
